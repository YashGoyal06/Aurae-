# AURAE | Premium Minimal Fashion

AURAE is a high-end, minimalist clothing brand website built with React, Vite, Tailwind CSS, and Framer Motion. The project focuses on a cinematic user experience, smooth animations, and premium fashion aesthetics.

## Features

- **Premium Loader**: Brand name reveal with subtle glow and smooth transition.
- **Home Page**: Fullscreen hero section, featured categories, and bestsellers.
- **Collection Pages**: Responsive grids for Men and Women with hover effects.
- **Product Detail**: High-resolution image gallery, size selector, and "Add to Bag" interaction.
- **Micro-animations**: Seamless page transitions and scroll-reveal effects using Framer Motion.
- **Modern UI**: Clean typography (Inter & Playfair Display) and a minimal color palette.

## Tech Stack

- **Framework**: React (Vite)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Routing**: React Router DOM

## Project Structure

```text
src/
├── components/       # Reusable UI components (Navbar, Loader, ProductCard, Footer)
├── pages/            # Page-level components (Home, Collection, ProductDetail)
├── data/             # Static data (Products, Categories)
├── styles/           # Custom CSS and Tailwind layers
└── App.jsx           # Main application logic and routing
public/
└── images/           # High-quality generated fashion imagery
```

## Setup & Run

1. `npm install`
2. `npm run dev`
